## 프로젝트 소개

UIKit 기반 코드를 이해할 수 있는 역량을 기르기 위해 진행한 넷플릭스 클론 코딩입니다.</br>

최근에는 SwiftUI를 많이 사용한다고 하지만, 기존에 오래된 서비스들은 UIKit으로 구성되어 있는 경우가 많아 </br>
코드를 알아볼 수 있을 정도의 역량은 지녀야 한다고 생각했습니다.</br>
